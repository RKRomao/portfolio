Linha de Apoio ao Caloiro

11/07/2024
Web design Web development User Experience User Interface

A Linha de Apoio ao Caloiro é uma plataforma pessoal, desenvolvida com dedicação e propósito, para servir como um guia completo para estudantes universitários recém-chegados — os famosos caloiros. O objetivo é simples mas impactante: facilitar a adaptação à vida académica com informação útil, dicas práticas e suporte personalizado.

Tecnologias:

HTML5, CSS3, JavaScript, Express.js, Node.js, MongoDB

Funcionalidades:

Responsivo design, Sistema de categorias, Busca por tags, Sistema de FAQ, Integração com GitHub

Descrição Detalhada:

A Linha de Apoio ao Caloiro é uma plataforma web desenvolvida para ajudar caloiros universitários a se adaptarem à vida académica. O projeto foi criado com o intuito de fornecer um guia completo e fácil de usar, que inclui: - Informações sobre os primeiros dias na universidade - Dicas para se organizar academicamente - Guia de sobrevivência ao primeiro ano - Recursos úteis para estudantes - FAQ sobre a vida universitária O projeto foi desenvolvido com foco em UX/UI para garantir que os alunos possam encontrar as informações necessárias de forma rápida e intuitiva.
