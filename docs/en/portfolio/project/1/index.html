Freshman Support Line

11/07/2024
Web design Web development User Experience User Interface

The Freshman Support Line is a personal platform, developed with dedication and purpose, to serve as a comprehensive guide for newly arrived university students — the freshmen. The goal is simple yet impactful: to facilitate adaptation to academic life with useful information, practical tips, and personalized support.

Technologies:

HTML5, CSS3, JavaScript, Express.js, Node.js, MongoDB

Features:

Responsive design, Category system, Tag search, FAQ system, GitHub integration

Detailed Description:

The Freshman Support Line is a web platform developed to help university freshmen adapt to academic life. The project was created to provide a complete and easy-to-use guide, which includes: - Information about the first days at university - Tips to organize yourself academically - First-year survival guide - Useful resources for students - FAQ about university life The project was developed with a focus on UX/UI to ensure that students can find the necessary information quickly and intuitively.
